# Spotify Clone

A front-end Spotify clone project built with JavaScript, HTML, and CSS.

This project was created as part of my practice while learning front-end development. It includes a music app style interface and basic functionality. The project is not fully finished yet because I had some difficulties with the API integration.

## Features

- Spotify-inspired user interface
- Built with HTML, CSS, and JavaScript
- Practice project for working with APIs
- Basic music app structure

## Project Status

This project is currently incomplete.

I was able to build the main structure and parts of the functionality, but I had difficulties completing the API integration. I am keeping this project in my portfolio as part of my learning process and progress in JavaScript.

## Technologies Used

- HTML
- CSS
- JavaScript
- API integration

## How to Run

1. Download or clone the repository
2. Open the project folder in VS Code
3. Run the project with Live Server or open the `index.html` file in your browser
